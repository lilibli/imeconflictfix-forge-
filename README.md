# ImeConflictFix-Forge

一个专为 Minecraft Forge 1.21.10+ 版本设计的轻量级模组，旨在解决在游戏内输入文字时，因输入法导致的按键冲突问题（如候选词跳跃、选词切换物品栏等）。灵感来源于 IMBlocker。

⚠️ **声明**：本项目所有代码均由 AI 辅助生成，本人主要负责测试与效果验证。

## ✨ 功能

*   **阻止空格键跳跃**：在文本输入框激活时，避免按下空格确认候选词时角色意外跳跃。
*   **阻止数字键切换快捷栏**：避免输入法选词时，数字键触发快捷栏切换。
*   **阻止 Tab 键切换玩家列表**：防止在输入时，Tab键切换玩家列表而打断输入。
*   **智能检测**：仅在输入法处于激活状态且文本框打开时拦截按键，不影响正常游戏操作。

## 🎮 适用场景

| 界面 | 空格 | 数字键 | Tab |
| :--- | :---: | :---: | :---: |
| 聊天 / 命令 | ✅ 拦截 | ✅ 拦截 | ✅ 拦截 |
| 告示牌编辑 | ✅ 拦截 | ✅ 拦截 | - |
| 书与笔 | ✅ 拦截 | ✅ 拦截 | - |
| 铁砧 | ✅ 拦截 | ✅ 拦截 | - |
| 搜索框 | ✅ 拦截 | ✅ 拦截 | - |
| 普通游戏 | ❌ 放行 | ❌ 放行 | ❌ 放行 |

## ⚙️ 技术信息

*   **Minecraft 版本**: 1.21.10 - 1.21.11
*   **Forge 版本**: 60.1.0 及以上
*   **Java 版本**: 21
*   **模组加载器**: Forge (非 NeoForge)

## 🔨 构建指南

如果您想从源码构建此模组，请确保您已安装 JDK 21，然后在项目根目录下执行：

./gradlew build

构建成功的 JAR 文件将位于 `build/libs/` 目录下。
