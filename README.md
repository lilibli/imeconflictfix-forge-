# imeconflictfix-forge-
用ai（deepseek-V4pro）仿照IMBlocker（mc模组）写的一个mc输入法冲突修复模组
所有代码均由ai完成，本人只测试代码完成效果

## 功能

- 在文本输入时**阻止空格键**触发跳跃（避免输入法确认候选词时意外跳跃）
- 在文本输入时**阻止数字键**切换快捷栏（避免输入法选词时切换物品栏）
- 在文本输入时**阻止 Tab 键**切换玩家列表
- **智能检测**输入法活动状态，仅在必要时拦截按键

## 版本信息

- Minecraft: **1.21.10**
- Forge: **60.1.0**
- Java: **21**
- 模组加载器: **Forge** (非 NeoForge)
